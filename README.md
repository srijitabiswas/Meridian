# Meridian

> A premium AI-powered Data Automation Platform landing page built for a frontend hackathon.

---

## Overview

Meridian is a modern SaaS landing page for an AI-powered data automation platform, developed entirely from scratch as part of a frontend hackathon.

The project emphasizes **clean architecture, responsive design, performance optimization, accessibility, SEO, and smooth native animations**, while strictly adhering to the competition constraints.
